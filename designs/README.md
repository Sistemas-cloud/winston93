# Diseños Pencil – Winston93

Esta carpeta es para archivos **.pen** de [Pencil](https://pencil.so) con las pantallas del proyecto.

## Pantallas del proyecto (rutas ↔ archivos código)

| Pantalla en Pencil   | Ruta en la web     | Archivo en el proyecto        |
|----------------------|--------------------|-------------------------------|
| Inicio (Home)        | `/`                | `src/pages/index.tsx`         |
| Contacto             | `/contacto`        | `src/pages/contacto.tsx`       |
| Oferta Educativa     | `/oferta-educativa` | `src/pages/oferta-educativa.tsx` |
| Primaria             | `/primaria`        | `src/pages/primaria.tsx`      |
| Programas            | `/programas`       | `src/pages/programas.tsx`     |
| Secundaria           | `/secundaria`      | `src/pages/secundaria.tsx`    |
| Servicios en línea   | `/servicios-en-linea` | `src/pages/servicios-en-linea.tsx` |
| Winston Life         | `/winston-life`    | `src/pages/winston-life.tsx`  |

## Cómo usar

1. En Pencil, **guarda el documento** de pantallas en esta carpeta, por ejemplo:  
   `designs/winston93-screens.pen`
2. Cada **frame** del .pen corresponde a una pantalla; puedes editar su diseño dentro del frame (1440×900 px).
3. Los cambios que hagas en Pencil puedes llevarlos al código en `src/pages/` y `src/components/`.

## Tamaño de pantalla de diseño

- **Desktop:** 1440 × 900 px (tamaño usado en los frames del .pen).
